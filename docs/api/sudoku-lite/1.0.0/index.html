Installation

Prerequisites

  • Node.js v18 or higher
  • npm v9 or higher

Quick Start

# Clone the repository
git clone https://github.com/AcceleratorMX/sudoku-lite.git
cd sudoku-lite

# Install dependencies
npm install

# Start the development server
npm run dev

Open http://localhost:5173 in your browser.

How to Play

  1. Enter your player name on the start screen
  2. Select a difficulty level (Easy / Medium / Hard / Expert)
  3. Click Start Game to begin
  4. Fill cells with numbers 1â€“9 so that each row, column, and 3Ã—3 box contains no duplicates
  5. Track your moves, mistakes, and time in real-time
  6. Complete the puzzle to see your final score and grade

Privacy Policy

Sudoku Lite operates entirely on the client side. No data is sent to any server. All game data (player name, progress, settings, scores) is stored locally in your browser's localStorage. See the full Privacy Policy for details.
